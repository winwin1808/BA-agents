# Changelog Package — Mar 18, 2026

This package contains merchant-facing changelog drafts for the three BSS Shopify apps plus a reusable template.

## Included files

- `changelog-bss-b2b-lock-login-password.md`
- `changelog-bss-b2b-request-a-quote-quick.md`
- `changelog-bss-b2b-wholesale-solution.md`
- `changelog-template.md`

## Editorial notes

- App names were aligned with the current Shopify App Store branding.
- Lock and Solution are written in version-based format.
- Quote is written in date-based format because the source notes did not include version numbers.
- The Quote draft includes one `Date to confirm` section for updates that did not include a publish date in the source.
- The Solution draft preserves duplicate version labels because the source notes used overlapping version numbers on some releases.

## Suggested publishing flow

1. Confirm the publish date for the `Date to confirm` section in the Quote draft.
2. If needed, split multi-item same-day entries into separate site posts.
3. Copy each release entry into the changelog site with the app's existing visual format.
