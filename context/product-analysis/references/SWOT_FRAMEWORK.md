# SWOT Framework

Use SWOT to assess internal capability and external market context before making strategic decisions.

## Internal

- Strengths
- Weaknesses

## External

- Opportunities
- Threats

## How to Use It

1. List evidence-based items
2. Prioritize the top 3-5 items per quadrant
3. Convert the matrix into strategy implications:
   - `SO`: Use strengths to capture opportunities
   - `ST`: Use strengths to defend threats
   - `WO`: Build capabilities to capture opportunities
   - `WT`: Mitigate weaknesses and threats

## Output Requirement

- Prioritized SWOT matrix
- Strategy implications using SO/ST/WO/WT
- Clear recommendation from the matrix, not just the matrix itself
