# Gibson Biddle DHM Framework

## Overview

DHM = Delight, Hard to Copy, Margin-Enhancing.

Use this framework to assess whether an initiative creates customer value, defensibility, and sustainable economics.

## D - Delight

- What user value and "aha" moment does this create?
- Evidence: adoption, retention, NPS, qualitative feedback

## H - Hard to Copy

- What moat exists? Examples: network effects, switching costs, brand, process, data
- How long would competitors need to replicate?

## M - Margin-Enhancing

- Revenue impact: pricing, expansion, retention
- Cost impact: support, operations, delivery efficiency
- Unit economics impact: LTV/CAC, gross margin

## Scorecard

Score each initiative from 1-5 for each pillar:

- `12-15`: High strategic priority
- `8-11`: Conditional
- `<8`: Low priority

## Output Requirement

- Score and rationale for Delight
- Score and rationale for Hard to Copy
- Score and rationale for Margin-Enhancing
- Supporting evidence for each score
- Overall priority interpretation
