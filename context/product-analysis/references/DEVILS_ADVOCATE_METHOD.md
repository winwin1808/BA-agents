# Devil's Advocate Method

Use this framework to stress-test a strategic choice before commitment.

## Process

1. State the choice clearly: `We should X because Y`
2. Challenge the choice using five lenses:
   - Assumptions
   - Risks
   - Opportunity cost
   - Evidence gaps
   - Alternatives
3. Defend or revise the choice
4. Add mitigations and decision triggers

## Output Requirement

- Top objections by severity
- Decision response for each objection
- Mitigation owner and due date

## Working Notes

- Focus on the highest-risk objections first
- If the choice cannot survive the top objections, revise it rather than defending a weak plan
- Separate evidence-based objections from speculative objections
