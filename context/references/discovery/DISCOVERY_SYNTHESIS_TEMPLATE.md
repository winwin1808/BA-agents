# Discovery Synthesis Template

Use this template when synthesizing discovery inputs such as customer interviews, support tickets, sales notes, research sessions, usage signals, or feedback reviews.

## 1. Input Summary

- Data sources
- Time window
- Coverage by module and persona

## 2. Evidence

- Signal table with: source, quote or fact, module, persona, severity

## 3. Inference

- Pattern candidates
- Confidence level
- Competing explanations

## 4. Opportunity Tree

- Outcome
- Opportunities
- Hypotheses

## 5. Prioritization

- Impact
- Confidence
- Effort
- Risk
- Recommended top 3 tests

## 6. Actions

- Owner
- Due date
- Metric linkage

## Working Notes

- Separate raw evidence from interpretation
- Mark confidence explicitly when evidence is thin or conflicting
- Keep hypotheses testable
- Prefer evidence-backed prioritization over intuition-only ranking
- End with concrete next actions, not just observations
